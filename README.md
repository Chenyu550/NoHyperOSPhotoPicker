# NoHyperOSPhotoPicker

 拦截并禁用澎湃OS魔改的照片视频/文件选择器，并恢复安卓原生选择器

 适用于基于Android 16的HyperOS 3

 在3.0.303.0上测试通过

## 介绍

 在金标联盟的推动下，小米在澎湃OS3中新增了“安全访问框架”，但是这个框架存在诸多问题，如不能显示没有后缀的文件，不能选择第三方来源（如系统相册、MT管理器【间接导致了无法访问Android/data等私有目录】、OneDrive等），本模块可以将这些框架恢复原生行为

 本质是通过 `pm` 禁用澎湃魔改选择器的活动并启用原生的活动，但是被禁用的选择器活动会不定时死灰复燃，故本模块诞生（如果你有更好的实现方法，欢迎提出来）

## 说明

 所谓的“安全访问框架”，其实就是在谷歌PhotoPickerAPI等的基础上用自己的UI二次包装而成，是原生就有的东西。

 因此恢复原生行为并不意味着访问图片和文件会变得“不安全”，应用仍然只能获取你勾选的文件，无法获取其余的文件。
